# 1. PYTORCH:
#    Instale primeiro PyTorch com suporte CUDA adequado:
#    pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
#
# 2. XFORMERS:
#    Para melhor performance, instale xformers:
#    pip install xformers
#
# 3. DLIB (opcional):
#    No Windows, pode ser necessário instalar via conda:
#    conda install -c conda-forge dlib
#
# 4. PHONEMIZER:
#    Requer espeak instalado no sistema:
#    - Ubuntu/Debian: sudo apt-get install espeak-ng
#    - Windows: baixar de https://github.com/espeak-ng/espeak-ng/releases
#    - macOS: brew install espeak-ng
#
# 5. FFMPEG:
#    Necessário para processamento de vídeo:
#    - Ubuntu/Debian: sudo apt-get install ffmpeg
#    - Windows: baixar de https://ffmpeg.org/download.html
#    - macOS: brew install ffmpeg
#
# 6. INSTALAÇÃO COMPLETA:
#    pip install -r requirements.txt
#
# 7. INSTALAÇÃO MÍNIMA (sem opcionais):
#    grep -v "^#" requirements.txt | grep -v "opcional" | pip install -r /dev/stdin
